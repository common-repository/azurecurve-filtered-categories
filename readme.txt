== Description ==
Creates a new Categories sidebar widget which allows categories to be included/excluded. A link to a categories page listing all categories can be configured to be displayed; a shortcode [fc] can be used on this page to display categories list. 

This plugin is multisite compatible.

This plugin supports language translations. If you want to translate this plugin please sent the .po and .mo files to wordpress.translations@azurecurve.co.uk for inclusion in the next version (full credit will be given). The .pot fie is in the languages folder of the plugin and can also be downloaded from the plugin page on http://development.azurecurve.co.uk.

== Installation ==
To install the plugin copy the <em>azurcurve-tag-cloud</em> folder into your plug-in directory and activate it. 

A Settings page allows all settings to be configured; deploy the widget to the sidebar of your choice to see the filtered categories.
